CSS transform e rotate

A propriedade transform no CSS permite Aplicar transformações a um elemento, como rendimento, rotação, inclinação ouo movimentação. É uma ferramenta poderosa, que permite criar efeitos visuais sem a necessidade de scrips.

O que e rotate?

S função rotate dentro da propriedade transform é usada para girar um elemento em torno de um ponto de origem. A rotação é especificada em graus deg, mas também pode usar outras unidades como radianos rad ou frações de volta turn.

Exemplos básicos:

transform: rotate(45deg) - Esre Código gira um elemento em 45 graus no sentindo horário.
